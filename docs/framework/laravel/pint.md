# Laravel pint

- GitHub: https://github.com/laravel/pint
- Docs: https://laravel.com/docs/master/pint
- Packagist: https://packagist.org/packages/laravel/pint

## 安装

```shell
composer require laravel/pint --dev
```

## 配置

```json
{
  "preset": "laravel",
  "exclude": []
}
```

## 使用

```shell
pint
```
