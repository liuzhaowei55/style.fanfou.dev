husky

  • typicode/husky
  • husky docs

安装​

  • npm
  • Yarn
  • pnpm
npm install husky --save-dev
yarn add husky --dev
pnpm add husky --save-dev

初始化​

npx husky init

设置 hooks​

echo "npm test" > .husky/pre-commit

提交 .husky 文件变化​

git add .husky/pre-commit

完全卸载​

npm uninstall husky

rm -rf .husky

git config --unset core.hooksPath
