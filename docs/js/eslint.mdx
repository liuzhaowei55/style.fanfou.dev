

# ESLint

- [ESLint Homepage](https://eslint.org/)

## 安装

```bash npm2yarn
npm install --save-dev eslint @umijs/lint
```

## 配置

.eslintrc.js

```bash
cat <<EEE > .eslintrc.js
module.exports = {
  extends: [require.resolve('@umijs/lint/dist/config/eslint')],

  rules: {
    // your rules
  },
};
EEE
```

.eslintignore

```ignore title=".eslintignore"
*.css
```

## 使用

```bash
eslint "src/**"
```
