

# ESLint

- [ESLint Homepage](https://eslint.org/)

## 安装

```bash npm2yarn
npm install --save-dev eslint @umijs/fabric
```

配置

```bash
cat <<EEE > .eslintrc.js
const fabric = require('@umijs/fabric');

module.exports = {
  extends: [require.resolve('@umijs/fabric/dist/eslint')],

  rules: {
    // your rules
  },
};
EEE
```

## 使用

```bash
eslint "src/**"
```
